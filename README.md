## 📌 Project Description

This project is a **Library Management System** developed using HTML, CSS, and JavaScript. It allows users to manage books efficiently by adding, searching, issuing, and returning books. The system includes role-based access (Admin & Student), due date tracking, and a dashboard to monitor library activities.

---

## 🚀 Features

* User Login (Admin / Student)
* Add and Manage Books
* Search and Filter Books
* Issue / Return System
* Due Date & Fine Alert
* Dashboard Overview
* Local Storage Support

---

## 🛠️ Technologies Used

HTML, CSS, JavaScript
